POISON CANDY CHALLENGE


คุณกำลังพบเจอกับความเป็นพิษ

ที่ไม่สามารถหลีกเลี่ยงได้จากเพื่อนหรือไม่

แคมเปญชาเลนจ์นี้้จะพาคุณเข้าสู่การพบเจอพิษที่หลักเลี่ยงไม่ได้

โดยการสุ่มลูกอมที่คุณไม่สามารถเลือกได้ แล้วพิษนั้น

จะตรงกับเพื่อนของคุณไหม ? ลองสุ่มเลย !!

เพื่อนรักชอบโกหก ( ลิลลี่ )

เพื่อนรักเห็นแก่ตัว ( เซลล์ )

เพื่อนรักหลงตัวเอง ( แน็ต )

เพื่อนรักชอบล้ำเล้นความเป็นส่วนตัว( ไพร์ช )

เพื่อนรักขี้นินทา ( กอส )

HOW TO PLAY


วิธีการเล่น

1.คลิก "เขย่า" เพื่อสุ่มลูกอมต่างๆออกมาจากกล่อง

2.สามารถกดเข้าไปเพื่ออ่านรายละเอียดเพิ่มเติมของลูกอม
